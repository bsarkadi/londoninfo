About

London Info Board is a collection of web applications created for the Hired.com London Hackathon by Balazs Sarkadi.

The website is built on the Bootstrap responsive framework and it supports all device sizes. It contains the following applications:

Tube Facility Finder: Find facilities such as cash machines, car parks or photo booths on the lines of the London Underground! Choose a line and a facility, and the app returns the stations with the number of facilities (if available), shows the stations on the embedded Google map and collects a photo gallery of each station from Flickr (using the Flickr photo search API), which appears in a pop-up if you click on the station's name. There's also a news ticker about travel incidents at stations.
The data is provided by TfL Open Data. Plugins used:

  • jQuery Marquee (news ticker)
  • Ekko Lightbox (photo gallery)
  • xml2json (XML to JSON conversion)
  • Google Maps

    • Weather Gauges: See the latest Air Pollution data, and 3-hourly weather data for the next 5 days by London boroughs displayed on gauges! The gauges are scaled and colored to match the displayed data. Air pollution data is provided by the London Air API while weather data comes from the Met Office DataPoint. Plugins used:

  • JustGage (gauges)
